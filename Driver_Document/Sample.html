FB3 EID_INVALIDDATA Invalid data! -Check the data passed to the method or property.
FB4 EID_INVALIDTIME Invalid time delta! Check the Time values passed to the methods in the ISMICMotion Interface. For Class 4, also see ISMICMotion::WriteClock. For Class 5, 6 or 9, also see ISMICMotion::WriteSegmentTime.
Get Status FB5 EID_INVALIDTIMEDELTA Motor(x): Invalid time delta! Check the Time values passed to the methods in the ISMICMotion Interface. For Class 4, also see ISMICMotion::WriteClock. For Class 5, 6 or 9, also see ISMICMotion::WriteSegmentTime.
Get Status FB6 EID_INVALIDPOSDELTA Motor(x): Invalid Position Delta! Check the position values passed to the methods in the ISMICMotion Interface. Also see ISMICMotion::WritePosition.
Get Status FB7 EID_INTDATASPACE Motor(x): Invalid data space! Check the program in the SmartMotor.
Get Status FB8 EID_BUFFOVERFLOW Motor(x): Buffer overflow! Do not send data continually faster than the time delta.
Get Status FB9 EID_BUFFUNDERFLOW Motor(x): Buffer underflow! Try a larger Time Delta. Shut down non-essential tasks and applications and try again.
Get Status FBA EID_HMFAILED Motor(x): Host Mode failed! Check and make sure that the specified motor's shaft can move and it is not overloaded.
Get Status FBB EID_GETSTATUS Motor(x): Cannot get status! Check:
- Serial connection to the motors.
- Power supply to the motors.
FBC EID_SAMPLERATES The motors have different sample rates! Replace with compatible SmartMotor.
FBD EID_NOTSUPPORTED Motor(x): Host Mode not supported! Replace the specified SmartMotor with a newer version supporting Host Mode. Make sure all motors are the same class.
FBE EID_NOTINIT Host Mode not initialized! Initialize the Host Mode and ISMICMotion Interface using the ISMICMotion::Initialize method before using any other method or property.
FBF EID_NOMEM Out of memory! Check the available memory. Shut down non-essential tasks and applications and try again.
FC0 EID_GETALL Cannot get data from more than one motor! Make sure the address of the SMIMotor object is not zero. Make sure you are not writing data over CAN to address zero. This is due to the handshaking required.
FC1 EID_NOMORE No more errors/warnings! Check the values of ISMIErrors::NoOfErrors and ISMIErrors::NoOfWarnings.
FC2 EID_OPENFILE Cannot open the file specified! Check the existence of the specified file.
FC3 EID_OPENFILEWRITE Cannot open the file to write! Make sure the file is not open and locked by other application. Make sure there is enough space on the destination disk.
FC4 EID_NOEEPROM The motor has no EEPROM! Check the SmartMotor EEPROM.
FC5 EID_INVALIDMODE Invalid operation mode! Check the usage of ISMICMotion::EndHostMode method.
- For Class 4, make sure in all previous calls to the ISMICMotion::WriteClock method, the address is zero.
- For Class 5, 6 or 9, make sure in all previous calls to the ISMICMotion::WriteSegmentTimemethod, the address is zero.
FC7 EID_FILETOOLONG File too long! Reduce the compiled user program size.
FC9 EID_DOWNLOADON Cannot download programs while motor is on! Turn off the motor before attempting to download or set the flag MDF_TURNOFF.
FCA EID_VERSIONMISMATCH File version has a major difference with motor version! Compile the program using the correct language file.
Eth. Com. FCB EID_SOCKET Error communicating over the network! Try opening the Ethernet port again.
FCC EID_DOWNLOADRCKS The program was NOT downloaded successfully! Download the program again. A problem occurred during the download.
Eth. Com. FCD EID_NETERROR A network error occurred! Verify the selected Ethernet adapter is the desired one.
NOTE: Only when opening the port.
If a network communication error is detected.
Can occur due to these reasons:
- using a specified adapter with a bad IP address format
- not able to open a UDP socket for transmitting or receiving
- setting the socket as non blocking
- sending a broadcast out socket, unable to open the detected IP address

-Try unplugging and plugging in the USB cable to the PC again.

Get Status FEB EID_EXCESSIVETEMP Motor(x): Excessive Temperature Fault! Motor may be undersized or ambient temperature is too high. Consider adding heat sinks or forced air cooling to the system.
Get Status FEC EID_EXCESSIVEPOS Motor(x): Excessive Position Fault! Increase error limit(EL), decrease load or make movement less aggressive.
Get Status FED EID_VELOCITYLIMIT Motor(x): Velocity Limit! Increase the velocity limit(VL) or decrease the delta between commanded positions or increase the segment time.
Get Status FEE EID_DEDTLIMIT Motor(x): DE/DT Limit! Increase the derivative error limit(DEL).
Get Status FEF EID_HWPOSLIMIT Motor(x): Historical Positive H/W Limit! Verify wiring and location of limit. If limit is not used change I/O to general input.
Get Status FF0 EID_HWNEGLIMIT Motor(x): Historical Negative H/W Limit! Verify wiring and location of limit. If limit is not used change I/O to general input.
Get Status FF1 EID_SWPOSLIMIT Motor(x): Historical Positive S/W Limit! Increase the software limit(SLP) or disable the software limits(SLD).
Get Status FF2 EID_SWNEGLIMIT Motor(x): Historical Negative S/W Limit! Decrease the software limit(SLN) or disable the software limits(SLD).

FF4 EID_BADMODEL Invalid model format! Enter a valid model.
