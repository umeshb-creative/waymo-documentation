    System Tray

    Once the PC/Machine is restarted grpc service would be running automatically. To verify this there would be an icon with waymo logo in the system tray. On right click it will have few options which are given below.

    1. Start Node-red : User/Operator has to click on this to start the node-red server along with the Waymo UI
    2. Stop Node-red : User can stop the node-red except the Waymo UI
    3. Start : Initially this option would be disable since the service would be running. it will enable only if the service is stopped
    4. Stop : On click the gRPC service would be stopped
    5. Exit : On click the grpc service would be stoppped also the icon would be going way from the system tray. User/Operator has to start the service from Desktop icon.
    6. View Logs : User will be taken to the log folder on the system.
    7. (0)devices online : It will show the no of devices which are online along with Alias There will also be Node-red shortcut to run only the node-red server
